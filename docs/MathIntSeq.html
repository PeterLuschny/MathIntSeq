Mathematical Integer Sequences

Prefixes of a sequence number referring to the OEIS:

  • A: Single term: A[n]
  • L: List: initial segment of A of length n.
  • B: Bound: all A[n] ≤ bound.
  • C: C: Coroutine (channel) generating A.
  • G: Generator function which iterates over A.
  • T: Triangular array with n rows.
  • M: Matrix of a triangular array with n rows.

→ Sequence names are linked to the source.

BellTrans

The Bell transform transforms an integer sequence into an integer triangle; also known as incomplete Bell polynomials. Let $X$ be an integer sequence, then $$B_{n,k}(X) = \sum_{m=1}^{n-k+1} \binom{n-1}{m-1} X[m] B_{n-m,k-1}(X)$$ where $B_{0,0} = 1, B_{n,0} = 0$ for $n≥1, B_{0,k} = 0$ for $k≥1$.

  • BellTrans(n::Int, k::Int, X) = Belltrans(n, k, X)
